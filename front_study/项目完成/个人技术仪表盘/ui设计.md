### 2.2 左侧导航

- **宽度**：240px
    
- **样式**：深色玻璃态（backdrop-blur）或纯黑底
    
- **内容**：
    
    - 头像 + 简短介绍（如 "Code. Build. Share."）
        
    - 导航菜单（图标 + 文字）
        
    - 可加 GitHub / 掘金 外链图标
        

### 2.3 主内容区布局（关键）

按 **从上到下、从左到右** 的信息层级排列：

1. **GitHub 热力图**（最显眼，证明活跃度）
    
    - 占满一整行
        
    - 支持年份切换
        
2. **第二行：左右两栏**
    
    - **左 60%**：平台数据聚合（多卡片横向排列或网格）
        
    - **右 40%**：智能待办 + 番茄钟
        
3. **第三行：技术标签云 / 雷达图**
    
    - 宽度 100%，与 GitHub 卡片对齐

### 3.1 GitHub 热力图卡片

- 用 **类似 GitHub 原生贡献日历** 的方块矩阵
    
- 深色背景 + 绿色层级（#0e4429 → #39d353）
    
- 右侧加简单统计：总贡献数、最长连续
    

### 3.2 平台数据聚合卡片

每个平台一张小卡片，包含：

- 平台图标（掘金 / CSDN / 知乎 / 思否）
    
- 关键指标：文章数 / 粉丝 / 点赞 / 阅读量
    
- 荧光绿强调数值
    
- 鼠标悬停微动效
    

> 初期可写死数据，第二阶段走 API 或手动后台填写

### 3.3 智能待办卡片

- 今日目标（带 checkbox）
    
- 里程碑进度条（如 “距 1k Star 还差 45”）
    
- 简易番茄钟（开始 / 暂停 / 重置）
    

### 3.4 技术标签云

- 标签按熟练度 / 使用频率决定字体大小
    
- 荧光绿 hover
    
- 点击标签 → 跳转知识花园并自动筛选该分类笔记
## 四、UI 风格细化（极客风）

| 元素   | 设计                                    |
| ---- | ------------------------------------- |
| 主背景  | `#0A0A0F` 或 `#0D1117`                 |
| 卡片背景 | `rgba(22, 27, 34, 0.8)` 带细边框          |
| 主色调  | `#00FF99`（荧光绿）                        |
| 辅助色  | `#3B82F6`（蓝，用于链接或强调）                  |
| 字体   | JetBrains Mono / Inter（代码用 Fira Code） |
| 边框   | 细 1px `#2D2D3A`                       |
| 阴影   | 轻微发光或低透明度阴影                           |
| 动效   | 卡片 hover 边框变绿 + 微抬升                   |
